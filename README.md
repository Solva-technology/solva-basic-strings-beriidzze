# Упражнения на работу со строками в Python

Эта серия заданий поможет вам освоить базовые и продвинутые операции со строками: срезы, индексацию и использование встроенных методов.

## Цель

Вам предстоит реализовать несколько функций в файле `main.py`, каждая из которых проверяет конкретный навык работы со строками.

## Инструкция

### Шаг 1: Настройте окружение

```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

### Шаг 2: Выполните задания в `main.py`

Откройте `main.py` и реализуйте следующие функции:

1.  **`get_first_and_last_char(s)`**: Должна возвращать строку, состоящую из первого и последнего символа исходной строки.

2.  **`get_middle_chars(s)`**: Должна возвращать все символы строки, кроме первого и последнего.

3.  **`reverse_string(s)`**: Должна возвращать строку в обратном порядке.

4.  **`is_palindrome(s)`**: Должна проверять, является ли строка палиндромом (читается одинаково в обоих направлениях), и возвращать `True` или `False`.

5.  **`count_vowels(s)`**: Должна считать и возвращать количество гласных букв (a, e, i, o, u) в строке.

6.  **`remove_spaces(s)`**: Должна удалять все пробелы из строки.

7.  **`capitalize_words(s)`**: Должна делать заглавной первую букву каждого слова в строке.

### Шаг 3: Запустите тесты

Для каждой функции подготовлены тесты. Запустите их, чтобы проверить свои решения:

```bash
pytest
```

Если все верно, тесты пройдут успешно. В противном случае вы увидите подробный отчет об ошибках.
